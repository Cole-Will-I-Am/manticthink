Questions, answered

What is SƎER?

A free, private web UI for Ollama models. Chat with Ollama Cloud models using your own API key, or connect to the Ollama running on your own machine — with streaming replies, charts, in-browser code execution, and GitHub context.

Do I need an account?

No account, no signup. Paste your own Ollama API key — or use local mode with the Ollama on your computer, which needs no key at all.

Where are my conversations stored?

Only in your browser. Our server never stores your messages or your key — and in local mode, requests never reach our servers at all. The one exception: if you tap Share on an AI Debate, that debate is saved on our servers so the short link works for others — only what you explicitly share. See the privacy policy.

Can I chat with models on my own computer?

Yes — choose “Use local Ollama” above and your browser connects straight to your local instance. A zero-install web UI for local LLMs.

Which models can I use?

Any model your Ollama key or local instance offers — gpt-oss, Qwen, DeepSeek, Llama, and more. You curate your own list in the model picker.
